# Contributing

Thanks for helping improve Codex Usage Pill.

## Development setup

1. Use Windows 10 or 11 with .NET Framework 4.x.
2. Fork and clone the repository.
3. Create a focused branch.
4. Run `./build.ps1` from PowerShell.
5. Test `dist/CodexUsagePill.exe --preview` and verify the real overlay with Codex desktop.

Keep changes small and avoid adding dependencies unless the benefit clearly outweighs the cost. Never commit authentication files, tokens, browser data, Codex account details, or proprietary OpenAI assets such as OpenAI Sans font files.

Pull requests should explain what changed, why it changed, and how it was verified.
